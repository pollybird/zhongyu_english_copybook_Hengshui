# 钟毓英语衡水体字帖生成器

## 项目简介

钟毓英语衡水体字帖生成器是一款专业的英语书写练习工具，专注于生成标准的衡水体英语字帖，帮助学生和英语学习者提高书写水平。

## 功能特点

- **多种生成模式**：支持描红、抄写、描红+抄写、纯字帖四种模式
- **多种线格类型**：提供四线三格和单横线两种线格样式
- **丰富的自定义选项**：
  - 字体大小和字间距调整
  - 位置偏移设置
  - 字帖标题自定义
  - 头部字段自定义（班级、姓名、学号等，最多支持3个字段）
- **PDF导出**：将生成的字帖导出为PDF文件，方便打印
- **分页预览**：实时预览生成效果，支持上一页/下一页导航
- **快捷键支持**：PageUp/PageDown键快速翻页

## 系统要求

- Windows 10/11 64位系统
- Python 3.8+
- PyQt6

## 安装说明

### 方法一：使用安装程序（推荐）

1. 下载 `钟毓英语衡水体字帖生成器安装程序.exe`
2. 双击运行安装程序，按照提示完成安装
3. 安装完成后，可从开始菜单或桌面快捷方式启动程序

### 方法二：从源码运行

1. 克隆项目到本地
2. 安装依赖：`pip install -r requirements.txt`
3. 运行程序：`python main.py`

## 使用说明

1. **输入内容**：在左侧「输入内容」区域输入要生成字帖的英文文本
2. **调整设置**：在「调整设置」面板中设置字体大小、字间距、位置偏移等参数
3. **选择线格**：在「线格类型」中选择适合的线格样式
4. **选择模式**：在「生成模式」中选择生成模式
5. **设置标题**：在「标题设置」中输入字帖标题
6. **自定义头部**：在「自定义头部」中添加或删除自定义字段
7. **预览效果**：在右侧预览区域查看生成效果，使用上一页/下一页按钮或PageUp/PageDown键导航
8. **导出PDF**：点击「导出PDF」按钮将字帖导出为PDF文件

## 项目结构

- `main.py`：主程序文件，包含主窗口和核心逻辑
- `renderers.py`：线格渲染器模块，负责不同线格类型的绘制
- `utils.py`：工具函数模块，包含页数计算等辅助功能
- `dialogs.py`：对话框模块，包含用户协议对话框
- `style.py`：样式定义模块，定义界面样式
- `font/`：字体文件目录，包含舒窈英文衡水体字体
- `app_icon.ico`：应用图标

## 文件关联

程序支持 `.zyecb` 格式的工程文件，双击该格式文件可直接用本程序打开。

## 常见问题

### 字体加载失败
- 确保 `font/` 目录下存在 `舒窈英文衡水体.ttf` 文件
- 如字体加载失败，程序会自动使用系统默认字体

### 导出PDF失败
- 确保目标目录有写入权限
- 检查输入内容是否为空
